Linux Commands Tutorial

This tutorial gives a brief introduction to beginner level Linux commands which are needed when dealing with file management and terminal in Linux.

  1. Listing file (ls) - If you want to see a list of files on your UNIX or LINUX system, use 'ls' command. It shows the files/directories in your current working directory. You can use 'ls -R' command to show all the files in the subdirectories as well. It is important to note here that this command is case sensitive, so typing 'ls -r' will result in an error.
  2. The 'cat' command is used to display text file. It can also be used for copying, combining and creating new text files. To create a new file:
    1. cat > filename
    2. Add content
    3. Press enter and new file will be created
  3. The syntax to merge two files using 'cat' command is 'cat file1 file2 > new file'
  4. The 'rm' command removes any file from the system without confirmation. To remove a file use the syntax 'rm filename'
  5. To move a file use the command 'mv filename new_file_location' and for renaming a file 'mv filename newfilename'
  6. Directories can be created on Linux using the command 'mkdir directoryname'. This will create a sub directory in your current working directory, which is usually your home directory.
  7. You can also create more than one directories at once, using 'mkdir dir1 dir2 dir3'.
  8. To remove a directory, use the following command 'rmdir dirname'. Ensure that the directory you wish you remove is empty before deletion.
  9. To rename a directory 'mv directoryname newdirectoryname' is used.

  1. The 'Man' command stands for manual which is standard reference for Linux OS. It is similar to help command found in other software.
  2. The 'history' command shows all the commands that you have used in the past for the current terminal session. This can help you refer to the old commands you have entered and re-used them in your operations again.
  3. The 'clear' command clears all the clutter on the terminal and gives you a clean window to work on, just like when you launch the terminal.
